ONLY FOR WINDOWS
----------------

This directory contains libintl and libiconv from the gnuwin32 project.
The binaries were downloaded and added to this project on January 23, 2016
from the following URL:
  gnuwin32.sf.net/packages/libiconv.htm
  gnuwin32.sf.net/packages/libintl.htm

The packages are under the LGPL and/or GPL 2.0 licenses.
We distribute the dll files as part of the Windows package.
